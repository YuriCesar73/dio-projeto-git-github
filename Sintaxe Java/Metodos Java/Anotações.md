Anotações e convenções sobre Métodos:

Definição: Todas as ações das aplicações são consideradas métodos.

Critérios de nomeação:
	Ser nomeado como verbo
 	Seguir o padrão camelCase
	Métodos tem uma única responsabilidade, ou seja, não é recomendavel que um método faça mais de uma ação.


Pontos de reflexão ao criar um método:
	Qual a proposta inicial do método?
 	Qual o tipo de retorno esperado após executar o método?
